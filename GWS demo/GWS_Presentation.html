2 / 6 Concept

什麼是 Antigravity？

Antigravity 不是一個被動的「聊天對話框」，而是一位具備**「調度權」**的 AI 中心指揮官。

透過建立標準作業流程 (SOP/Skills)，它能自動串接我們平常用的各種工具（例如：Google Drive、試算表、NotebookLM 知識庫），把人的腦力從「整理資料」解放出來，專注於「決策」。

Tech Concept

科普一下：什麼是 MCP？

剛剛我們提到了 MCP 生態系，這聽起來很技術，但其實概念非常簡單。你完全可以把它想像成**「軟體界的 Type-C 轉接線」**。

以前的 API (各種專屬充電線)

以前 AI 要接 Google Drive、接 ERP、接財務庫，工程師必須為每個系統開發「專屬的連接線 API」。這就像你家的抽屜躺著凌亂的 Micro-USB、Lightning 等舊線材，買新設備就要買新線，極度耗時且難以擴充維護。

🆚

現在的 MCP (Type-C 萬用快充線)

MCP (Model Context Protocol) 是一個全球統一的開源標準接頭。現在，只要給 AI 大腦配備這條「萬用 Type-C」，無論是筆記本還是資料庫，一插上去瞬間就能讀懂你的數據，完全不需重新客製化！
